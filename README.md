# ASCII_art_generator
**An image to ascii art convertor program**
This is a command line application written in pure **C** that converts images into **ASCII ART** using low-level processing Techniques.

The project focuses on:
-Raw image memory handling
-Grayscale & coloured ASCII conversion
-Terminal-based rendering using ANSI escape codes

## ^.^ Features
-Load image from file path(supports quoted Windows paths)
-Convert image to ACII art
 -Grayscale ASCII
 -Colored ASCII (ANSI true-color)
-Menu-based CLI
-Fast & light-weight(pure C + 'stb_image.h')
-Spatial downsampling for terminal rendering.

>Best viewed in Windows Terminal or Linux terminal with true-color support

**conversion technique**
RGB-> grayscale by ITU-R BT.601
